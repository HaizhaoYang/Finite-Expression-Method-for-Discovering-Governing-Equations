# Finite Expression Methods for Discovering Physical Laws from Data

The data and code for the paper [Jiang, Zhongyi, Chunmei Wang, and Haizhao Yang. "Finite expression methods for discovering physical laws from data." arXiv preprint arXiv:2305.08342 (2023).](https://arxiv.org/abs/2305.08342)

#### Burgers’ equation with constant coefficients
- Run [burgers_cc_FEX.py](FEX/burgers_cc_FEX.py)

#### Burgers’ equation with varying coefficients
- Run [burgers_vc_FEX.py](FEX/burgers_vc_FEX.py)

#### 2D Hopf normal form
- Run [Hopf_FEX.py](FEX/Hopf_FEX.py)

#### Johnson-Mehl-Avrami-Kolmogorov (JMAK) nonlinear equation
- Run [JMAK_FEX.py](FEX/JMAK_FEX.py)
